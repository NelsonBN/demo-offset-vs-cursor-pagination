# Demo - Cursor Pagination x Offset Pagination

## SQL Demo

```bash
docker compose up -d postgres pgweb
# or
docker compose -f docker-compose.sql.yml up
```
> Data explorer: `http://localhost:8081`



## NoSQL Demo

```bash
docker compose up -d mongodb mongo-express
# or
docker compose -f docker-compose.nosql.yml up
```
> Data explorer: `http://localhost:8081`
